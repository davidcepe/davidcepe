Introduction to C++

C++ is a powerful, high-performance programming language used for system/software development, game development, and competitive programming. Start with basic programs to understand syntax, input/output, and structure.

🎥 Tutorial: C++ Full Course for Beginners

Variables & Data Types

Variables store data. Examples: int age = 16;, double price = 19.99;, char grade = 'A';. Learn about int, float, double, char, bool, and string.

🎥 Tutorial: C++ Variables Tutorial

Functions

Functions are reusable code blocks. Example: int add(int a, int b){ return a+b; }. Functions can take parameters and return values, organizing your code efficiently.

🎥 Tutorial: C++ Functions Tutorial

Loops

Loops repeat code: for(int i=0;i<5;i++){ std::cout << i << std::endl; }. C++ supports for, while, and do...while loops.

🎥 Tutorial: C++ Loops Tutorial

Arrays & Vectors

Arrays store fixed-size data: int arr[5] = {1,2,3,4,5};. Vectors are dynamic arrays: std::vector<int> v = {1,2,3};. Loop through them to process elements.

🎥 Tutorial: C++ Arrays & Vectors Tutorial

Pointers

Pointers store memory addresses: int x = 10; int *ptr = &x;. They allow efficient memory management and dynamic allocation.

🎥 Tutorial: C++ Pointers Tutorial

Classes & Objects

Classes define objects: class Person { public: string name; int age; };. Objects are instances of classes: Person p; p.name = "Alex";. Encapsulation and constructors are key concepts.

🎥 Tutorial: C++ Classes Tutorial

Inheritance

Inheritance allows one class to acquire properties of another: class Student: public Person { int grade; };. Polymorphism and virtual functions are important for OOP.

🎥 Tutorial: C++ Inheritance Tutorial

Exception Handling

Handle runtime errors with try { } catch(...) { }. Example: try{ int x = arr[5]; } catch(std::exception &e){ std::cout << e.what(); }.

🎥 Tutorial: C++ Exception Handling Tutorial
